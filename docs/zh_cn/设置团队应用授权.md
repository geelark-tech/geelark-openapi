## 接口说明

授予团队应用权限，含位置权限等全部权限，仅对新安装应用生效。

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/auth/status`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 团队应用id | 497652752864775437 |
| appAuth | 是 | integer | app授权 0关闭授权；1打开授权 | 1 |

## 请求示例

```json
{
 "id": "497652752864775437",
 "appAuth": 1
}
```

## 响应示例

```json
{
 "traceId": "886A92FCBE9B7A52A7F583FCBD2BF6A8",
 "code": 0,
 "msg": "success"
}
```

## 错误码

请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
