## 接口说明

查询代理

## 请求URL

- `https://openapi.geelark.cn/open/v1/proxy/list`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| page | 是 | integer | 页码，最小为1 | 1 |
| pageSize | 是 | integer | 每页多少条数据，最小为1，最大为100 | 1 |
| ids | 否 | array[string] | 代理id列表 | 参考请求示例 |

## 请求示例

```json
{
	&quot;page&quot;: 1,
	&quot;pageSize&quot;: 1,
	&quot;ids&quot;: [
		&quot;493188072704313353&quot;
	]
}
```

## 响应体数据说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| total | integer | 数据总数 |
| page | integer | 页码 |
| pageSize | integer | 每页多少条数据 |
| list | array[ProxyListItem] | 代理信息项列表 |

### ProxyListItem 代理信息项

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 代理id |
| serialNo | int | 代理序号 |
| scheme | string | 代理类型，socks5，http，https |
| server | string | 代理地址 |
| port | integer | 代理端口 |
| username | string | 代理用户名 |
| password | string | 代理密码 |
