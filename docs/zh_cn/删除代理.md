## 接口说明

删除代理

## 请求URL

- `https://openapi.geelark.cn/open/v1/proxy/delete`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| ids | 是 | array[string] | 代理id列表，最多传100个 | 参考请求示例 |

## 请求示例

```json
{
	"ids": [
		"493188072704313353"
	]
}
```

## 响应体数据说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| totalAmount | integer | 处理总数，传入重复的id不计数 |
| successAmount | integer | 处理成功数 |
| failAmount | integer | 处理失败数 |
| failDetails | array[FailDetail] | 失败代理信息 |

### FailDetail 失败代理信息

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 代理id |
| code | integer | 错误码，参考失败细节错误 |
| msg| string | 错误信息，参考失败细节错误 |

### 失败细节错误

| 错误码 | 错误信息 |
| - | - |
| 40005 | proxy not found |
| 40010 | proxy binds to the environment |
