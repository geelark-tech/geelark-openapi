## 接口说明

代理检测接口

## 请求URL

- `https://openapi.geelark.cn/open/v1/proxy/check`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| detect_type | 是 | string | IP查询渠道，仅支持 IP-API 或者 IP2Location | IP2Location |
| proxy_type | 是 | string | 代理类型，仅支持 socks5/http/https  | socks5 |
| server | 是 | string | 主机 | 185.162.130.86 |
| port | 是 | int | 端口号 | 11000 |
| username | 否 | string | 代理账号 | username |
| password | 否 | string | 代理密码 | pass |

## 请求示例

```json
{
	"detect_type": "IP2Location",
	"proxy_type": "socks5",
	"server": "185.162.130.86",
	"port": 10000,
	"username": "username",
	"password": "pass"
}
```

## 响应体数据说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| detectStatus | bool | 是否检测成功 |
| message | string | 检测失败后的原因 |
| outboundIP | string | 出口ip |
| countryCode | string | 出口ip所属国家代码 |
| countryName | string| 出口ip所属国家名称 |
| subdivision | string| 出口ip所属州/省 |
| city | string| 出口ip所属城市|
| timezone | string| 出口ip所属时区 |
| isp | string| 出口ip所属isp |

## 响应示例

```json
	}

```
