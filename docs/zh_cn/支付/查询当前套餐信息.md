## 接口说明

查询当前套餐信息，此接口调用频率：1次/min

## 请求URL

- `https://openapi.geelark.cn/open/v1/pay/plan/info`

## 请求方法

- POST

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| plan | integer   | 订阅类型: 0 Base, 1 Pro |
| profiles  | integer    | 环境数 |
| monthlyRental  | integer    | 包月设备数 |
| parallels  | integer    | 并发数 |
| expirationTime  | integer    | 套餐到期时间 |
| monthlyFee  | float    | 月结费用 |
| availableProfiles  | integer    | 可用环境数 |
|availableMonthlyRentals|integer|可用包月设备数|
