## 接口说明

查询账户余额，此接口调用频率限制为 10/min

## 请求URL

- `https://openapi.geelark.cn/open/v1/pay/wallet`

## 请求方法

- POST

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| balance | float   | 余额 |
| giftMoney  |  float   | 赠送金 |
| availableTimeAddOn | int   | 时长包余额 |
