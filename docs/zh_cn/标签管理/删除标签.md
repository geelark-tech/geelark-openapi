## 接口说明


- 删除对应标签


## 请求 URL


- `https://openapi.geelark.cn/open/v1/tag/delete`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型          | 说明             | 示例         |
| ------ | ---- | ------------- | ---------------- | ------------ |
| ids    | 是   | array[string] | 删除标签 id 列表 | 参考请求示例 |


## 请求示例


```json
{
  &quot;ids&quot;: [&quot;528989565877224448&quot;, &quot;528953724308030464&quot;]
}
```


## 响应示例


```json
{
  &quot;traceId&quot;: &quot;ACEB0CFEB887F99CB989BC9D9FF92BBC&quot;,
  &quot;code&quot;: 0,
  &quot;msg&quot;: &quot;success&quot;,
  &quot;data&quot;: {
    &quot;totalAmount&quot;: 2,
    &quot;successAmount&quot;: 1,
    &quot;failAmount&quot;: 1,
    &quot;failDetails&quot;: [
      {
        &quot;code&quot;: 43022,
        &quot;id&quot;: &quot;528953724308030464&quot;,
        &quot;msg&quot;: &quot;tag not found&quot;
      }
    ]
  }
}
```


## 响应体数据说明


| 参数名        |     类型           |     说明     |
| ------------- | ------------------ | ------------ |
| totalAmount   | integer            | 请求删除总数 |
| successAmount | integer            | 成功总数     |
| failAmount    | integer            | 失败总数     |
| failDetails   | array[FailDetails] | 失败部分信息 |


### failDetails 失败信息 &lt;FailDetails&gt;


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 错误码       |
| id           | string     | 标签 id      |
| msg          | string     | 错误信息     |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明       |
| ------ | ---------- |
| 43022  | 标签不存在 |
