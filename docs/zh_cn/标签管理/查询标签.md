## 接口说明


- 获取标签信息


**标签颜色列表参考创建标签接口**


## 请求 URL


- `https://openapi.geelark.cn/open/v1/tag/list`


## 请求方法


- POST


## 请求参数


### 分页参数


| 参数名       | 必选   |     类型    |     说明                             | 示例 |
| ------------ | ------ | ----------- | ------------------------------------ | ---- |
| page         | 是     |   integer   | 页码，最小为 1                       | 1    |
| pageSize     | 是     |   integer   | 每页多少条数据，最小为 1，最大为 100 | 10   |


### 查询参数


| 参数名 | 必选 | 类型          | 说明         | 示例         |
| ------ | ---- | ------------- | ------------ | ------------ |
| ids    | 否   | array[string] | 标签 id 列表 | 参考请求示例 |
| names  | 否   | array[string] | 标签名字列表 | 参考请求示例 |
| colors | 否   | array[string] | 标签颜色列表 | 参考请求示例 |


## 请求示例


### 不携带查询条件


```json
{
  &quot;page&quot;: 1,
  &quot;pageSize&quot;: 5
}
```


### 根据 ids 查询


```json
{
  &quot;page&quot;: 1,
  &quot;pageSize&quot;: 5,
  &quot;ids&quot;: [&quot;528989565910778880&quot;, &quot;528989565894198272&quot;]
}
```


### 根据 names 查询


```json
{
  &quot;page&quot;: 1,
  &quot;pageSize&quot;: 5,
  &quot;names&quot;: [&quot;tagRed&quot;, &quot;tagWhite&quot;]
}
```


### 根据 colors 查询


```json
{
  &quot;page&quot;: 1,
  &quot;pageSize&quot;: 5,
  &quot;colors&quot;: [&quot;blue&quot;, &quot;red&quot;]
}
```

## 响应体数据说明


| 参数名       |     类型          |     说明     |
| ------------ | ----------------- | ------------ |
| total        | integer           | 总数         |
| page         | integer           | 页码         |
| pageSize     | integer           | 分页大小     |
| list         | array[Tag]        | 标签列表     |
| failDetails  | array[FailDetail] | 失败信息     |


### list 标签列表 &lt;tag&gt;


| 参数名 | 类型   | 说明     |
| ------ | ------ | -------- |
| id     | string | 标签 id  |
| name   | string | 标签名   |
| color  | string | 标签颜色 |


### failDetails 失败信息 &lt;FailDetail&gt;


| 参数名 | 类型    | 说明                                      |
| ------ | ------- | ----------------------------------------- |
| code   | integer | 错误码                                    |
| type   | integer | 失败类型：1-标签id 2-标签名称 3-标签颜色 |
| param  | string  | 失败参数                                  |
| msg    | string  | 失败信息                                  |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明           |
| ------ | -------------- |
| 43022  | 标签不存在     |
| 43023  | 标签颜色不存在 |
| 43024  | 标签名字不存在 |
