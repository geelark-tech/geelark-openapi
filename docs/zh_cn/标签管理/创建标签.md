## 接口说明


- 创建指定**名字，颜色（可选）**的标签
- 支持批量
- 创建时不选择颜色则默认为**白色（white）**


### color 列表


- `white`
- `red`
- `blue`
- `green`
- `yellow`
- `purple`


## 请求 URL


- `https://openapi.geelark.cn/open/v1/tag/add`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型           | 说明         | 示例         |
| ------ | ---- | -------------- | ------------ | ------------ |
| list   | 是   | array[TagItem] | 标签数据列表 | 参考请求示例 |


### list 数据列表&lt;TagItem&gt;


| 参数名 | 必选 | 类型   | 说明     | 示例                     |
| ------ | ---- | ------ | -------- | ------------------------ |
| name   | 是   | string | 标签名字，最长30个字符 | tag                      |
| color  | 否   | string | 标签颜色 | white，详情见 color 列表 |

## 响应体数据说明


| 参数名         |     类型              |     说明     |
| -------------- | --------------------- | ------------ |
| totalAmount    | integer               | 请求创建总数 |
| successAmount  | integer               | 成功总数     |
| failAmount     | integer               | 失败总数     |
| successDetails | array[SuccessDetails] | 成功部分信息 |
| failDetails    | array[FailDetails]    | 失败部分信息 |


### successDetails 成功信息 &lt;SuccessDetails&gt;


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| id           | string     | 标签 id      |
| name         | string     | 标签名       |
| color        | string     | 标签颜色     |


### failDetails 失败信息 &lt;FailDetails&gt;


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 失败码       |
| name         | string     | 失败标签名   |
| msg          | string     | 失败信息     |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明                              |
| ------ | --------------------------------- |
| 43020  | 标签名为空                        |
| 43021  | 标签名已存在                      |
| 43023  | 标签颜色不支持，请查看 color 列表 |
