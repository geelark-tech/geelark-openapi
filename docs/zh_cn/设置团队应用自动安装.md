## 接口说明

设置团队应用自动安装，开启自动安装后，应用会在云手机启动后安装。

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/setStatus`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 团队应用id | 497652752864775437 |
| status | 是 | integer | 是否要自动安装，0否，1是 | 1 |
| installGroupIds | 否 | array[string] | 允许安装的环境组，默认为所有环境组，0代表未分组 | ["497652752864775437"] |

## 请求示例

```json
{
 "id": "497652752864775437",
 "status": 1
}
```

## 响应示例

```json
{
 "traceId": "886A92FCBE9B7A52A7F583FCBD2BF6A8",
 "code": 0,
 "msg": "success"
}
```

## 错误码

请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
