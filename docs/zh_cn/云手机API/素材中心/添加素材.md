## 接口说明
添加素材


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/create`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| url | 是 | string | 请参考[上传临时文件到GeeLark](https://open.geelark.cn/api/upload-getUrl) | 参考请求示例|
| tagsId | 否 | array[string] | 标签id | 参考请求示例 |
| fileName | 否 | string | 文件名，最多200字符  | 参考请求示例 |


## 请求示例
```json
{
 &quot;url&quot; : &quot;https://material.geelark.cn/client-img/banner0903_cn.gif&quot;,
 &quot;tagsId&quot; : [&quot;569577509402731994&quot;,&quot;569577514586891738&quot;],
 &quot;fileName&quot;:&quot;a.jpg&quot;
}
```


## 响应示例

```json
{
 &quot;traceId&quot;: &quot;ADD9A7489BB2198DBC0FB37082684CB0&quot;,
 &quot;code&quot;: 0,
 &quot;msg&quot;: &quot;success&quot;,
 &quot;data&quot;: {
 &quot;id&quot;: &quot;570606523940605924&quot;,
 &quot;failDetails&quot;: [
 {
 &quot;id&quot;: &quot;5695775094027319941&quot;,
 &quot;code&quot;: 43022,
 &quot;msg&quot;: &quot;tag not found&quot;
 }
 ]
 }
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- 
| id | string | 素材id |
| failDetails | failDetails | 失败信息 |

### failDetails 失败信息 &lt;FailDetails&gt;

| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 错误码       |
| id           | string     | 标签 id      |
| msg          | string     | 错误信息     |

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 60001 | 超出素材库最大容量 |
| 60003 | 非法的链接，需要使用上传到GeeLark的资源链接 |
| 60004 | 文件格式不支持 |
| 43022  | 标签不存在 |
| 40005  | 资源不存在，请检查url的资源是否可用 |
