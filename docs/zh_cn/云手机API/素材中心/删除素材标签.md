## 接口说明
删除素材标签


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/tag/del`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| ids | 是 | array[string] | 标签id | 参考请求示例 |


## 请求示例
```json
{
    "ids" : ["570461738663674391"]
}
```


## 响应示例

```json
{
  "traceId": "ACEB0CFEB887F99CB989BC9D9FF92BBC",
  "code": 0,
  "msg": "success",
  "data": {
    "totalAmount": 2,
    "successAmount": 1,
    "failAmount": 1,
    "failDetails": [
      {
        "code": 43022,
        "id": "528953724308030464",
        "msg": "tag not found"
      }
    ]
  }
}
```


## 响应体数据说明


| 参数名        |     类型           |     说明     |
| ------------- | ------------------ | ------------ |
| totalAmount   | integer            | 请求删除总数 |
| successAmount | integer            | 成功总数     |
| failAmount    | integer            | 失败总数     |
| failDetails   | array[FailDetails] | 失败部分信息 |


### failDetails 失败信息 &lt;FailDetails&gt;


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 错误码       |
| id           | string     | 标签 id      |
| msg          | string     | 错误信息     |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明       |
| ------ | ---------- |
| 43022  | 标签不存在 |
