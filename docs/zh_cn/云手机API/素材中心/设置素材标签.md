## 接口说明
设置素材标签


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/tag/set`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| materialsId | 是 | array[string] | 素材id | 参考请求示例 |
| tagsId | 是 | array[string] | 标签id，每次打标签操作会删除原有的所有标签，仅保留新设置的标签 | 参考请求示例 |


## 请求示例
```json
{
 &quot;materialsId&quot; : [&quot;570457374221926935&quot;],
 &quot;tagsId&quot; : [&quot;570461738663674391&quot;]
}
```


## 响应示例

```json
{
    &quot;traceId&quot;: &quot;AC3ADD84BEA1D8D2A7DDAF8B90A8D2A8&quot;,
    &quot;code&quot;: 0,
    &quot;msg&quot;: &quot;success&quot;,
    &quot;data&quot;: {
        &quot;failDetails&quot;: [
            {
                &quot;id&quot;: &quot;5704573742219269351&quot;,
                &quot;code&quot;: 60005,
                &quot;msg&quot;: &quot;material not found&quot;
            },
            {
                &quot;id&quot;: &quot;5704617386636743911&quot;,
                &quot;code&quot;: 43022,
                &quot;msg&quot;: &quot;tag not found&quot;
            }
        ]
    }
}
```

### failDetails 失败信息 &lt;FailDetails&gt;


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 错误码       |
| id           | string     | 标签 id      |
| msg          | string     | 错误信息     |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明       |
| ------ | ---------- |
| 43022  | 标签不存在 |
| 60005  | 素材不存在 |
