## 请求URL

- `https://openapi.geelark.cn/open/v1/analytics/accounts/update`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 账号id | 565523829426802069 |
| account | 否 | string | 平台账号，最多64个字符 | myAccount |
| channel | 否 | int | 平台 | 0: TikTok 1:YouTube 2:Instagram |
| remark |否 | string | 备注 | myRemark |

## 请求示例

```json
{
    "id": "565523829426802069",
    "account": "abc"
}
```

## 响应示例

```json
{
    "traceId": "9D1A84D6A2A8F9A8A5CD9BA7B7E84281",
    "code": 0,
    "msg": "success"
}
```
