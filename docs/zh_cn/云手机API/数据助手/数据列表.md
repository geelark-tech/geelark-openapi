## 接口说明


仅能获取已经爬取的账号数据


## 请求 URL


- `https://openapi.geelark.cn/open/v1/analytics/data`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型             | 说明         | 示例         |
| ------ | ---- | ---------------- | ------------ | ------------ |
| page   | 是   | integer | 页码 | 1 |
| pageSize   | 是   | integer | 每页数量（1-100） | 10 |
| account   | 否   | string | 账号 | tk_acc |
| dataDate   | 否   | integer | 搜索日期时间戳 | 1764136841 |
| createdId   | 否   | string | 添加账号的用户id | 547236295441737181 |
| channel   | 否   | integer | 平台：0: TikTok 1:YouTube 2:Instagram | 1 |



## 请求示例


```json
{
	&quot;account&quot; : &quot;tk_acc&quot;,
	&quot;dataDate&quot; : 1764137986,
	&quot;createdId&quot; : &quot;497521349330210656&quot;,
	&quot;channel&quot; : 1,
	&quot;page&quot; : 1,
	&quot;pageSize&quot; : 10
}
```

## 响应体数据说明

| 参数名         |     类型              |     说明     |
| -------------- | --------------------- | ------------ |
| total    | integer               | 总数 |
| page  | integer               | 当前页面     |
| pageSize     | integer               | 当前每页条数     |

## 响应体数据说明 &lt;items&gt;

| 参数名         |     类型              |     说明     |
| -------------- | --------------------- | ------------ |
| id    | string               | 账号id |
| channel  | integer               | 渠道     |
| account     | string               | 账号     |
| playCount     | integer               | 播放量，-1 表示数据还没更新     |
| followerCount     | integer               | 粉丝数，-1 表示数据还没更新     |
| diggCount     | integer               | 点赞数据，-1 表示数据还没更新     |
| commentCount     | integer               | 评论数，-1 表示数据还没更新     |
| collectCount     | integer               | 收藏数 ，-1 表示数据还没更新     |
| shareCount     | integer               | 分享数，-1 表示数据还没更新     |
| dataDate     | integer               | 数据日期     |
| addAccDate     | integer               | 添加账号时间     |
| remark     | string               | 备注     |
| createdId     | string               | 添加账号的用户id     |
| username     | string               | 添加账号的用户名     |
