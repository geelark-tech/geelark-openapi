## 请求URL

- `https://openapi.geelark.cn/open/v1/analytics/accounts/delete`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| channel | 是 | int | 平台 | 0: TikTok 1:YouTube 2:Instagram |
| account | 是 | string | 账号名,最长64字符 | 参考请求示例 |

## 请求示例
```json
{
    "channel":0,
    "account":"xxxx"
}
```

## 响应示例

```json
{
	"traceId": "9D1A84D6A2A8F9A8A5CD9BA7B7E84281",
	"code": 0,
	"msg": "success"
}
```
