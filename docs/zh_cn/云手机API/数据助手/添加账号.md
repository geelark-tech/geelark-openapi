## 请求URL

- `https://openapi.geelark.cn/open/v1/analytics/accounts/add`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| channel | 是 | int | 平台 | 0: TikTok 1:YouTube 2:Instagram |
| accountsData |是 | array[accountsData] | 账号信息， 数组最大支持200个元素 |参考请求示例 |

#### accountsData
| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| account | 是 | string | 账号名,最长64字符 | 参考请求示例 |
| remark |否 | string | 备注信息 | 参考请求示例 |

## 请求示例
```json
{
    "channel":0,
    "accountsData": [
        {
            "account":"XXX",
            "remark":"remark xxx"
        }
    ]
}
```

## 响应参数

| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- | 
| bizCode | 是 | int | 业务状态码： 0全部成功；1表示当前存在账号数超出限制；2表示部分成功，失败部分超出限制。 |
| successCount |是 | int | 成功添加数 |
| failCount |是 | int | 失败添加数 |
| repeatCount |是 | int | 重复添加数 |

## 响应示例

```json
{
	"traceId": "9D1A84D6A2A8F9A8A5CD9BA7B7E84281",
	"code": 0,
	"msg": "success",
	"data": {
    	"bizCode": 0,
    	"successCount": 1,
    	"failCount": 0,
    	"repeatCount": 0
	}
}
```
