## 请求 URL


- `https://openapi.geelark.cn/open/v1/analytics/accounts/list`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型             | 说明         | 示例     |
| ------ | ---- | ---------------- | ------------ | ------------ |
| page   | 是   | integer | 页码 | 1 |
| pageSize   | 是   | integer | 每页数量（1-100） | 10 |
| account   | 否   | string | 账号 | tk_acc |
| channel | 否 | int | 平台，不填则为全部平台 | 0: TikTok 1:YouTube 2:Instagram |
| userAccount | 否 | string | 操作者账号 | abc@gmail.com |

## 请求示例
```json
{
    &quot;page&quot;:1,
    &quot;pageSize&quot;:10,
    &quot;channel&quot;:1
}
```



## 响应体数据说明

| 参数名         |     类型              |     说明     |
| -------------- | --------------------- | ------------ |
| total    | integer               | 总数 |
| page  | integer               | 当前页面     |
|items|array[item]|账号数据|

## 响应体数据说明 &lt;item&gt;

| 参数名         |     类型              |     说明     |
| -------------- | --------------------- | ------------ |
| id    | string               | 账号id |
| account     | string               | 账号     |
| channel  | int              | 平台 0: TikTok 1:YouTube 2:Instagram      |
| remark     | string               | 备注     |
| operator     | string               | 最后操作账号的用户名     |
| created_time     | int               | 添加时间     |
| updated_time     | int               | 修改时间     |
