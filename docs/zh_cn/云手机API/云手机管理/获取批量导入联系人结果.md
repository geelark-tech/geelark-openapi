## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/importContactsResult`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| taskId | 是 | string | 任务id | 参考请求示例 |


## 请求示例
```json
{
    "taskId": "528715748189668352"
}
```



## 响应示例

```json
{
    "traceId": "A62BBBF3A294487F9B49B9FFA0F84CA8",
    "code": 0,
    "msg": "success",
	"data": {
        "status": 2
    }
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| status | int | 任务状态：1执行中 2执行成功 3执行失败 |
