## 接口说明

- 发送短信到云手机，可以模拟发送一条短信至指定云手机。发送前，请先启动云手机

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/sendSms`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 |string | 云手机环境id（目前仅支持安卓12、13、14、15类型的机器） | 526209711930868736 |
| phoneNumber | 是 | string | 发送方手机号 | +17723504471 |
|text|是|string|短信内容|您的验证码为：62533|

## 请求示例

```json
{
 "id": "526209711930868736",
 "phoneNumber": "+17723504471",
 "text": "your tk code: 6666"
}
```


## 响应示例

```json
{
 "traceId": "9E681400B2983A5390F4B7C8BF1BF2B7",
 "code": 0,
 "msg": "success",
 "data": {}
}
```

| 错误码 | 说明 |
| --- | --- |
| 52001 | 此类型云手机不支持发送短信 |
