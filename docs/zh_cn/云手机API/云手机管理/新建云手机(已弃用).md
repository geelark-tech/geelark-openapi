## 接口说明

- 基础套餐一次仅支持创建一台云手机，Pro套餐支持批量新建
- 支持选择机型，请先调用云手机品牌列表接口获取支持的品牌机型信息

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/add`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| amount | 是 | integer | 创建总数，数量范围1-100 | 10 |
| androidVersion | 是 | integer | 云手机系统版本 | 1 |
| proxyId | 否 | string | 代理id。proxyId和proxyConfig必须传一个。若proxyId不为空，则使用proxyId，否则使用proxyConfig。 | 497548067550006541 |
| proxyConfig | 否 | proxyConfig | 代理信息 | 参考请求示例 |
| remark| 否 | string | 备注，最长1500个字符 | 12ABCDEF |
| groupName | 否 | string | 分组名，不存在自动新建，最长50个字符 | group1 |
| tagsName | 否 | array[string] | 标签名，不存在自动新建，单个标签最长30个字符 | 参考请求示例 |
|region|否|string|指定云手机所在的机房，可选参数：cn, sgp| cn|
|chargeMode|否|int|计费模式|0 按需计费，1 包月；默认为按需计费|
|language|否|string|云手机的语言|baseOnIP/default（default 就用英文）,如果此参数不传默认用英文|
| surfaceBrandName | 否 | string | 手机品牌，从品牌列表接口获取对应安卓版本的值，品牌型号要同时传值 | samsung |
| surfaceModelName | 否 | string | 手机型号，从品牌列表接口获取对应安卓版本的值，品牌型号要同时传值 | Galaxy S23 |
| netType | 否 | integer | 联网方式，0-Wi-Fi，1-移动网络，仅Andorid 12/Andorid 13/Andorid 15支持，默认为移动网络 | 0 |
| phoneNumber | 否 | string | 手机号，为空则自动生成 | +66817806147 |

### proxyConfig 代理信息

#### 静态代理

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| typeId | 是 | integer | 代理类型id | 1 |
| server | 是 | string | 代理服务器主机名 | server.com |
| port | 是 | integer | 代理服务器端口 | 1234 |
| username | 是 | string | 代理服务器账号 | user |
| password | 是 | string | 代理服务器密码 | password |

#### 动态代理

动态代理可以先在客户端设置，然后将useProxyCfg传true代表使用已设置的信息，不需要再传主机端口等信息。

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| useProxyCfg | 是 | bool | 是否使用已设置的代理 | true |
| typeId | 是 | integer | 代理类型id | 20 |
| protocol | 否 | integer | 代理协议类型。1代表socks5，2代表http。 | 1 |
| server | 否 | string | 代理服务器主机名 | server.com |
| port | 否 | integer | 代理服务器端口 | 1234 |
| username | 否 | string | 代理服务器账号 | user |
| password | 否 | string | 代理服务器密码 | password |
| country | 否 | string | 国家 | us |
| region | 否 | string | 区域 | alabama |
| city | 否 | string | 城市 | mobile |

### androidVersion 对应版本

- `1` : Android 10
- `2` : Android 11
- `3` : Andorid 12
- `4` : Andorid 13 
- `5` : Andorid 10 直播版 ---只支持包月计费模式
- `7` : Andorid 14 
- `8` : Andorid 15 

### typeId列表
#### 1. 静态代理列表
- `1` : socks5
- `2` : http
- `3` : https

#### 2. 动态代理列表
- `20` IPIDEA代理
- `21` IPHTML代理
- `22` kookeey代理
- `23` Lumatuo代理
