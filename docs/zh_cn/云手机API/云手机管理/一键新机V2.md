## 接口说明

一键新机v2接口，直接返回执行结果。

## 请求URL

- `https://openapi.geelark.cn/open/v2/phone/newOne`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 云手机id | 参考请求示例 |
| changeBrandModel | 否 | bool | 是否随机品牌型号，true：随机，false：保持不变；不填也是随机 | true |
| keepNetType | 否 | bool | 保留联网方式，默认为false，随机联网方式 | true |
| keepPhoneNumber | 否 | bool | 保留手机号码，默认为false，随机手机号 | true |
| keepRegion | 否 | bool | 保留地区，默认为false，跟随代理 | true |
| keepLanguage | 否 | bool | 保留语言，默认为false，使用英语 | true |

## 请求示例

```json
{
    "id": "528715748189668352"
}
```

## 响应示例

```json
{
 "traceId": "A62BBBF3A294487F9B49B9FFA0F84CA8",
 "code": 0,
 "msg": "success"
}
```

## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明 |
| --- | --- |
| 42001 | 云手机不存在 |
| 44004 | 今日创建环境数量达到上限 |
| 43005 | 云手机有任务正在执行 |
| 43006 | 云手机正在使用|
| 43015 | 该云手机不支持一键新机 |
| 45004 | 代理检测失败 |
| 43038 | 设备品牌型号已删除 |
