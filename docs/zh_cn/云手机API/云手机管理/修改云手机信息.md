## 接口说明
### 注意：请勿在调用启动云手机api的同时操作此接口

- 修改云手机信息
- 支持修改云手机名称
- 支持修改云手机备注
- 支持修改云手机标签
- 支持修改云手机代理配置
- 支持修改云手机分组
- 支持修改云手机计费方式
- 支持修改云手机手机号（需要关机后修改）

## 请求 URL


- `https://openapi.geelark.cn/open/v1/phone/detail/update`


## 请求方法


- POST


## 请求参数


| 参数名      | 必选 | 类型          | 说明                 | 
| ----------- | ---- | ------------- | -------------------- | ---- |
| id          | 是   | string        | 云手机 id            |
| name        | 否   | string        | 新云手机名称，最长100个字符         |
| remark      | 否   | string        | 新云手机备注，最长1500个字符         |
| groupID     | 否   | string        | 新云手机分组 id      |
| tagIDs      | 否   | array[string] | 新云手机标签 id 列表 |
| proxyConfig | 否   | Proxy         | 新云手机代理配置     |
| proxyId     | 否   | string        | 代理Id|
|chargeMode|否|int|计费模式：0 按需计费，1 包月；默认为按需计费|
| phoneNumber     | 否   | string        | 自定义手机号 |


### proxyConfig 代理信息

#### 静态代理

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| typeId | 是 | integer | 代理类型id | 1 |
| server | 是 | string | 代理服务器主机名 | server.com |
| port | 是 | integer | 代理服务器端口 | 1234 |
| username | 是 | string | 代理服务器账号 | user |
| password | 是 | string | 代理服务器密码 | password |

#### 动态代理

动态代理可以先在客户端设置，然后将useProxyCfg传true代表使用已设置的信息，不需要再传主机端口等信息。

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| useProxyCfg | 是 | bool | 是否使用已设置的代理 | true |
| typeId | 是 | integer | 代理类型id | 20 |
| protocol | 否 | integer | 代理协议类型。1代表socks5，2代表http。 | 1 |
| server | 否 | string | 代理服务器主机名 | server.com |
| port | 否 | integer | 代理服务器端口 | 1234 |
| username | 否 | string | 代理服务器账号 | user |
| password | 否 | string | 代理服务器密码 | password |
| country | 否 | string | 国家，取值请参考[动态代理国家代码列表](https://singapore-upgrade.geelark.cn/apiResource/proxy-country.txt)| us |
| region | 否 | string | 区域 ，请参考对应代理：[地区码](https://singapore-upgrade.geelark.com/apiResource/proxies-country-codes.txt) | 请根据对应代理填写 |
| city | 否 | string | 城市，请参考对应代理：[地区码](https://singapore-upgrade.geelark.com/apiResource/proxies-country-codes.txt)  | 请根据对应代理填写 |



### typeId列表
#### 1. 静态代理列表
- `1` : socks5
- `2` : http
- `3` : https

#### 2. 动态代理列表
- `21` IPHTML代理
- `22` kookeey代理
- `23` Lumatuo(BrightData)代理
- `24` Proxyma
- `25` SmartProxy代理
- `26` RolaIP代理
- `27` NodeMaven代理
- `29` KookeeyMobile代理

```json
{
  "id": "528086284158239744",
  "name": "api update",

  "proxyConfig": {

    "username": "username",
    "password": "password"
  },
 "proxyId": "528989565877355520"
```

## 响应示例


成功响应


```json
{
  "traceId": "B04B0843BD86D9589AB3BAB6A9EA3D92",
  "code": 0,
  "msg": "success"
}
```


标签列表中有部分标签不存在，则有 failDetails 数据；标签都不存在时，将直接返回请求错误，code 详见下方接口**错误码**


```json
{
  "traceId": "8B38AA778DBCD9519FB9B00D8A593DB3",
  "code": 0,
  "msg": "success",
  "data": {
    "failDetails": [
      {
        "code": 43022,
        "id": "52898956587728998",
        "msg": "tag not found"
      }
    ]
  }
}
```


## 响应体数据说明


| 参数名       |     类型           |     说明         |
| ------------ | ------------------ | ---------------- |
| failDetails  | array[FailDetails] | 添加标签失败信息 |


### failDetails 添加标签失败信息<FailDetasils>


| 参数名 | 类型    | 说明     |
| ------ | ------- | -------- |
| code   | integer | 错误码   |
| id     | integer | 标签 id  |
| msg    | string  | 错误信息 |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明                         |
| ------ | ---------------------------- |
| 42001  | 云手机不存在                 |
| 43022  | 标签不存在                   |
| 43032  | 分组不存在                   |
| 45003  | 代理地区不允许               |
| 45004  | 代理检测失败，请检查代理配置 |
| 45008 | 代理类型不允许 |
