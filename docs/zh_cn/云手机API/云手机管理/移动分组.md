## 接口说明
移动分组

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/moveGroup`


## 请求方法
- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| envIds | 是 | array[string] | 云手机id数组，最大支持100个元素 | 参考请求示例|
|groupId|是|string|分组id ，传&quot;0&quot;则移动至未分组|参考请求示例|

## 请求示例
```json
{
    &quot;envIds&quot;: [
        &quot;601876382020062634&quot;
    ],
    &quot;groupId&quot;: &quot;590711571886417453&quot;
}
```


## 响应示例

```json
{
    &quot;traceId&quot;: &quot;B3DAFF64A7BD493CB1169D94A22BFC8D&quot;,
    &quot;code&quot;: 0,
    &quot;msg&quot;: &quot;success&quot;
}
```
