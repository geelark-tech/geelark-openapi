## 接口说明

设置云手机联网方式

## 请求 URL


- `https://openapi.geelark.cn/open/v1/phone/net/set`


## 请求方法


- POST


## 请求参数


| 参数名      | 必选 | 类型          | 说明                 |
| ----------- | ---- | ------------- | -------------------- |
| id          | 是   | string        | 云手机 id            |
| netType | 是 | integer | 联网方式，0-Wi-Fi，1-移动网络，仅Andorid 12/Andorid 13/Andorid 15支持 |


## 请求示例


```json
{
  "id": "528086284158239744",
  "netType": 0
}
```


## 响应示例


```json
{
  "traceId": "B04B0843BD86D9589AB3BAB6A9EA3D92",
  "code": 0,
  "msg": "success"
}
```

## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明                         |
| ------ | ---------------------------- |
| 42001  | 云手机不存在                 |
