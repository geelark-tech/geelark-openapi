## 接口说明

- 获取云手机品牌列表

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/brand/list`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| androidVer | 是 | integer | 安卓版本，10-15 | 10 |

## 请求示例

```json
{
	&quot;androidVer&quot; : 10
}
```


## 响应示例

```json
{
	&quot;traceId&quot;: &quot;B0BA8FF29AB60B8ABF4E9A26BF08F7B9&quot;,
	&quot;code&quot;: 0,
	&quot;msg&quot;: &quot;success&quot;,
	&quot;data&quot;: [
		{
			&quot;surfaceBrandName&quot;: &quot;samsung&quot;,
			&quot;surfaceModelName&quot;: &quot;Galaxy S20+&quot;
		}
	]
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| surfaceBrandName | string | 手机品牌|
| surfaceModelName | string | 手机型号|

## 错误码


错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
