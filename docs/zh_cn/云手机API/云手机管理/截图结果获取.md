## 接口说明
查询云手机截图任务状态
请求截图后可在30分钟内通过该接口主动获取结果，过期后将获取失败

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/screenShot/result`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| taskId | 是 | string | 任务id | 参考请求示例 |


## 请求示例
```json
{
    "taskId": "528715748189668352"
}
```



## 响应示例

```json
{
    "traceId": "A62BBBF3A294487F9B49B9FFA0F84CA8",
    "code": 0,
    "msg": "success",
	"data": {
        "status": 2,
		"downloadLink": "https://zx-cloud-phone-pre.obs.cn-southwest-2.myhuaweicloud.com/envirFileExport/1851511129017700352/IMG_20241122160248.png?AccessKeyId=UFNIPAPFJX2MAGMFGRYZ&amp;Expires=1732264377&amp;Signature=dqV5JYzYDdm0wwAgkZIpDrs%2FL%2FE%3D"
    }
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| status | int | 0获取失败；1执行中；2执行成功；3执行失败 |
| downloadLink | downloadLink | 截图链接 |


## 错误码

错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
