## 接口说明
获取云手机状态

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/status`

## 请求方法

- POST

## 请求参数
### 查询参数
| 参数名       | 必选   |     类型       |    说明        | 示例       |
| ----------- | -------| -----------    |-----------    |----------- |
| ids         | 是     | array[string]  | 云手机id列表，最大限制100     | 参考请求示例       |

## 请求示例

```json
{
    "ids":[
        "123456ABCDEF",
        "123456ABCDEF",
        "123456ABCDEF",
        "123456ABCDEF"
    ]
}
```
## 响应体数据说明
| 参数名 | 类型 | 说明 |
| ----- | ----- | ----- |
| totalAmount | integer | 请求id总数 |
| successAmount | integer | 成功总数 |
| failAmount | integer | 失败总数 |
| successDetails | array[SuccessDetails] | 成功部分信息 |
| failDetails | array[FailDetails] | 失败部分信息 |

### successDetails 成功信息 &lt;SuccessDetails&gt;
| 参数名 | 类型 | 说明 |
| ----- | ----- | ----- |
| id | string | 成功云手机id |
| serialName | string | 成功云手机名称 |
| status | integer | 云手机状态码&lt;br/&gt; 0-已启动&lt;br/&gt; 1-启动中&lt;br/&gt; 2-已关闭&lt;br/&gt; 3-已过期 |

### failDetails 失败信息 &lt;FailDetails&gt;
| 参数名 | 类型 | 说明 |
| ----- | ----- | ----- |
| code | int | 失败码 42001: 云手机不存在 |
| id | string | 成功云手机id |
| msg | string | 失败信息 |
