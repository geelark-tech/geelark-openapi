## 接口说明
获取云手机列表


## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/list`

## 请求方法

- POST

## 请求参数

### 分页参数

| 参数名       | 必选   |     类型   |    说明    | 示例|
| ----------- | -------| -----------|----------- |----------- |
| page        | 否     |   integer  | 页码，最小为1，不填默认为1 | 1|
| pageSize    | 否     |   integer  | 每页多少条数据，最小为1，最大为100，不填默认为10 | 10|

### 查询参数 (空值则忽略)

| 参数名       | 必选   |     类型   |    说明    | 示例|
| ----------- | -------| -----------|----------- |----------- |
| ids | 否 | array[string] | 云手机id数组，数组最大长度为100，如果数组不为空，则 page pageSize 这两个参数不生效 | ["5213214343124321"] |
| serialName  | 否     |   string   | 云手机名称 | 测试 |
| remark      | 否     |   string   | 云手机备注 | 测试 |
| groupName   | 否     |   string   | 云手机分组名 | 测试分组 |
| tags | 否 | array[string] | 云手机标签名称列表 | 参考示例 |
|chargeMode|否|int|计费模式|0 按需计费，1 包月，不填则为全部计费模式|
| openStatus        |  否     |   integer  | 开关状态 | 0关闭；1开启 |
| proxyIds        |  否     |  array[string]   | 代理id数组，数组最大长度为10 |  ["5213214343124321"]  |
| serialNos        |  否     |  array[string]   | 序号数组，数组最大长度为100 |  ["238"]  |

## 请求示例
``` json
{
    "page":1,
    "pageSize":10,
    "serialName": "test",
    "remark":"",
    "groupName":"",
	"tags":[
		"tag1",
		"tag2"
	],
	"openStatus" : 1,
	"proxyIds": ["602943680722009770"],
	"serialNos": ["238"]
}
```


## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| total | integer   | 云手机总数  |
| page | integer   | 页码  |
| pageSize | integer   | 分页大小  |
| items | array[Phone]   | 云手机数组  |

### items 云手机数据 &lt;Phone&gt;

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | string   | 云手机id  |
| serialName  | string    | 云手机名称 |
| serialNo  | string    | 云手机序号 |
| group  | Group    | 云手机分组信息 |
| remark  | string    | 云手机备注 |
| status | int | 云手机状态&lt;br/&gt; 0-已启动&lt;br/&gt; 1-启动中&lt;br/&gt; 2-已关机 |
| tags  | array[Tag]    | 云手机标签数组 |
| equipmentInfo  | EquipmentInfo    | 云手机设备信息 |
| proxy | Proxy | 代理信息 |
|chargeMode|int|计费模式：0 按需计费，1 包月|
| hasBind | bool | 是否绑定了包月设备 |
| monthlyExpire | int | 包月过期时间，秒级时间戳 |
|rpaStatus|int|是否在执行rpa，1执行中，0未在执行|

#### group 分组信息 &lt;Group&gt;
| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 分组id |
| name | string | 分组名称 |
| remark | string | 分组备注 |

### tags 云手机标签 &lt;Tag&gt;

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| name | string | 云手机标签名称 |

### equipmentInfo 云手机设备信息 &lt;EquipmentInfo&gt;

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| countryName | string | 国家名，请参考[国家名称对照表](https://material.geelark.cn/t_region.xls) |
| phoneNumber | string | 手机号码 |
| enableSim | int | SIM卡状态 0未启用1 已启用 |
| imei | string | IMEI |
| osVersion | string | 系统版本 |
| wifiBssid | string | Wi-Fi MAC地址 |
| mac | string | 手机Wi-Fi MAC地址 |
| bluetoothMac | string | 蓝牙地址 |
| timeZone | string | 时区 |
|deviceBrand|string|品牌|
|deviceModel|string|型号|
|deviceName|string|设备名|
|netType|integer|联网方式，0-Wi-Fi，1-移动网络|
|language|string|云手机的语言|
|province|string|省，只有创建时候指定，此值才不为空|
|city|string|市，只有创建时候指定，此值才不为空|

#### proxy 代理信息 &lt;Proxy&gt;

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| type | string | 代理类型(socks5, http, https) |
| server | string | 代理地址 |
| port | int | 代理端口|
| username | string | 代理用户名 |
| password | string | 代理密码 |
