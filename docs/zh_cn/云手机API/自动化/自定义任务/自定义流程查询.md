##接口说明
- 获取当前账户下自定义RPA任务列表

## 请求URL

- `https://openapi.geelark.cn/open/v1/task/flow/list`


## 请求方法
- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 |
| --- | --- | --- | --- |
| page | 是 | integer | 页码，最小为1 |
| pageSize | 是 | integer | 每页多少条数据，最小为1，最大为100 |

## 请求示例
```json
{
	&quot;page&quot;: 1,
	&quot;pageSize&quot;: 1
}
```

## 响应体数据说明

| 字段名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| total | integer   | 数据总数  |
| page | integer | 页码 |
| pageSize | integer | 每页多少条数据 |
| items | array[TaskFlow] | 任务流程数组  |

### 任务流程 TaskFlow

| 字段名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | string   | 任务流程id |
| title | string   | 任务流程标题 |
| desc | string   | 任务流程描述 |
| params | array[string]   | 任务流程参数字段名 |
