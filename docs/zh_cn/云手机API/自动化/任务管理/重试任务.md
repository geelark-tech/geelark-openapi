## 接口说明

一个任务最多重试5次。
任务失败时，客户端创建的任务会自动重试2次，调用api创建的任务不会自动重试。
如果自动重试后任务还是失败，则可以调用该接口重试。
当任务处于以下状态时，可以调用接口重试。
- `任务失败`
- `任务取消`

## 请求URL

- `https://openapi.geelark.cn/open/v1/task/restart`

## 请求方法

- POST

## 请求参数

| 参数名       | 必选   |     类型   |    说明    |
| ----------- | -------| -----------|----------- |
| ids        | 是     |   array[string]  | 任务id数组 |

## 请求示例

```json
{
    "ids": ["123321", "456654"]
}
```

## 响应体数据说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| totalAmount | integer | 处理总数 |
| successAmount | integer | 处理成功数 |
| failAmount | integer | 处理失败数 |
| failDetails | array[FailDetail] | 处理失败细节 |

### FailDetail

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 任务id |
| code | integer | 错误码 |
| msg | string | 错误信息 |

## 错误码

响应体外层错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

### 单个任务处理错误码

| 错误码 | 说明 |
| --- | --- |
| 40005 | 环境已删除 |
| 48000 | 任务重试次数已达上限 |
| 48001 | 任务状态不允许重试 |
| 48002 | 任务不存在 |
| 48003 | 任务资源已过期 |
