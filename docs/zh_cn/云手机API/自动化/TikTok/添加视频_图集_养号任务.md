## 接口说明

- 创建养号任务，直接调用添加任务接口即可。
- 创建发视频及发图集任务，需要先将素材上传，再调用添加任务接口。
- 调用该接口创建的养号任务，不会自动重试。


## 请求URL

- `https://openapi.geelark.cn/open/v1/task/add`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| planName | 否 | string | 任务计划名称，不传则自动生成 | testAdd |
| remark | 否 |string| 备注，最多200字 | task |
| taskType | 是 | integer | 任务类型&lt;br&gt;1 发布视频&lt;br&gt;2 懒人养号&lt;br&gt;3 发布图集 | 3 |
| list | 是 | array | 任务参数数组，一次最多创建100个任务 | 参考请求示例 |

## 请求示例

### 示例一 懒人养号

```json
{

    &quot;taskType&quot;: 2,
    &quot;list&quot;: [
        {
            &quot;scheduleAt&quot;: 1718744459,
            &quot;envId&quot;: &quot;123456654321&quot;,
            &quot;action&quot;: &quot;search video&quot;,
            &quot;keywords&quot;: [&quot;hi&quot;],
			&quot;duration&quot;: 10
        }
    ]
}
```

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 41000 | 任务积分不足 |
| 43004 | 云手机已过期，请续费或升级套餐 |
| 41001 | 余额不足 |
| 43018 | 包月云手机没有绑定包月设备 |
| 48004 | 任务所需app不满足条件 |
