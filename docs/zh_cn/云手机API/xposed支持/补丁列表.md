## 接口说明

补丁列表，目前只支持Android 12系统、Android 13系统、Android 15系统

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/patch/list`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 否 | string | 云手机id | 1830906144634757120 |

## 请求示例

```json
{
 &quot;id&quot;: &quot;532930887177274368&quot;
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| list | []PatchInfo | 补丁信息|

### PatchInfo：补丁信息
| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | id | test |
| version | 是 | string | 版本 | 1753258366 |
| description | 是 | string | 描述 | 补丁测试 |
| owner | 是 | string | owner | unknown |
