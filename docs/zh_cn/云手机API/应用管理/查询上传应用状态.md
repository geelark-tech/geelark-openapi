## 接口说明

查询上传应用状态

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/upload/status`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| taskId | 是 | string | 任务id | 1830906144634757120 |

## 请求示例

```json
{
    &quot;taskId&quot; :  &quot;1830906144634757120&quot;
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| status | int | 状态 0正在上架；1上架成功；2上架失败；3审核不通过 |
| appName | string | 应用名  |
| appIcon | string | 应用图标  |
| appId | string | 应用id  |
| versionId | string | 应用版本id  |
