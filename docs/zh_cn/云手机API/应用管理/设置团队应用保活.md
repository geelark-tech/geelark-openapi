## 接口说明

开启或关闭团队应用保活。仅支持Pro套餐。
仅支持Android 12/13/14/15，且最多保活1个应用。
Android 12/13/15若在开启保活前已启动App，需重启App后生效。
Android 14须在开机状态下设置开启/取消保活后，再重启云手机才生效。

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/setKeepAlive`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 团队应用id | 497652752864775437 |
| opera | 是 | integer | 操作，0关闭，1打开 | 1 |

## 请求示例

```json
{
 &quot;id&quot;: &quot;497652752864775437&quot;,
 &quot;opera&quot;: 1
}
```

## 响应示例

```json
{
 &quot;traceId&quot;: &quot;886A92FCBE9B7A52A7F583FCBD2BF6A8&quot;,
 &quot;code&quot;: 0,
 &quot;msg&quot;: &quot;success&quot;
}
```
