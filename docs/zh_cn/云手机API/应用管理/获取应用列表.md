## 接口说明

获取应用列表

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/shop/list`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| key | 否 | string | 搜索关键字 | tiktok |
| getUploadApp | 否 | bool | 获取上传的应用 | true |
| page        | 是     |   integer  | 页码，最小为1 | 1|
| pageSize    | 是     |   integer  | 每页多少条数据，最小为1，最大为200 | 10|

## 请求示例

```json
{
 "key" : "tiktok",
 "getUploadApp" : false,
 "page" : 1,
 "pageSize" : 5
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| total | integer   | 总数  |
| page | integer   | 页码  |
| pageSize | integer   | 分页大小  |
| items | array[AmpAppInfo]   | 数据数组  |

### 应用信息 AmpAppInfo

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| appIcon | string   | 应用图标 |
| id | string    | 应用id |
| appName  | string    | 应用名 |
| appVersionList | array[AmpAppVersionInfo] | 应用版本信息|

### 应用版本信息 AmpAppVersionInfo

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | string    | 应用版本id |
| versionCode | integer | 应用版本码|
| versionName | string | 应用版本名|
