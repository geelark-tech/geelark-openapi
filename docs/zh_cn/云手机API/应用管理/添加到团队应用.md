## 接口说明

添加应用到团队应用，云手机启动后自动安装

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/add`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 应用id；请调用“获取应用列表”接口获取 | 1793552962123993090 |
| versionId | 是 | string | 版本id；请调用“获取应用列表”接口获取 | 1793552962140770305 |
| installGroupIds | 否 | array[string] | 允许安装的环境组，默认为所有环境组，&quot;0&quot; 代表未分组；请先调用查询分组接口获取分组id | [&quot;528715748189668352&quot;] |

## 请求示例

```json
{
 &quot;id&quot;: &quot;1793552962123993090&quot;,
 &quot;versionId&quot;: &quot;1793552962140770305&quot;,
 &quot;installGroupIds&quot;: [&quot;0&quot;]
}
```

## 响应示例

```json
{
 &quot;traceId&quot;: &quot;886A92FCBE9B7A52A7F583FCBD2BF6A8&quot;,
 &quot;code&quot;: 0,
 &quot;msg&quot;: &quot;success&quot;
}
```

## 错误码

请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
