## 接口说明

批量对已打开的云机做应用的操作

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/operation/batch`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| action | 是 | interger | 操作类型 1启动；2关闭；3重启；4安装；5卸载 | 1 |
| groupIds | 否 | array[string] | 分组id，操作指定分组下的已打开的云机，默认全部 | ["595681312937911830"] |
| packageName | 否 | string | 应用包名，启动/关闭/重启/卸载 推荐使用包名（packageName/versionId 二选一） | io.tm.k.drama |
| versionId | 否| string | 应用版本id，安装必传，可通过/app/teamApp/list接口获取 （packageName/versionId 二选一）|1793552962140770305 |

### 请求示例

启动
```json
{
    "action" : 1,
    "groupIds" : ["595681312937911830"],
    "packageName": "io.tm.k.drama"
}
```

安装
```json
{
    "action" : 1,
    "groupIds" : ["595681312937911830"],
    "versionId": "1793552962140770305"
}
```

## 响应体数据说明 
### items 
| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | integer   | 云手机id  |
| errCode | integer   | 错误码 1应用正在安装/卸载；2未安装对应的app  |

### 响应示例

```json
{
    "traceId": "A0598A2CA0802A98957B9E1F87EB9289",
    "code": 0,
    "msg": "success",
    "data": {
        "items": [
            {
                "id": "583502967211075086",
                "errCode": 2
            }
        ]
    }
}
```

## 错误码

错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
