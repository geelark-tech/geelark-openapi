## 接口说明

关闭应用

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/stop`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| envId | 是 | string | 云手机环境id | 1809135651036667904 |
| appVersionId | 否 | string | 应用版本id(应用版本id/应用包名两者必须有其一) | 1793552962140770305 |
|packageName| 否|string|应用包名(应用版本id/应用包名两者必须有其一) |com.zhiliaoapp.musically|

## 请求示例

```json
{
    &quot;envId&quot; : &quot;1809135651036667904&quot;,
    &quot;appVersionId&quot; : &quot;1793552962140770305&quot;
}
```


## 响应示例

```json
{
    &quot;traceId&quot;: &quot;123&quot;,
    &quot;code&quot;: 0,
    &quot;msg&quot;: &quot;success&quot;
}
```

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 42001 | 不存在对应的云手机 |
| 42002 | 云手机不处于运行状态 |
| 42005 | 未安装对应的app |
