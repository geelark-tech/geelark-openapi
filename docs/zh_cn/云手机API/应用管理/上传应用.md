## 接口说明

上传应用

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/upload`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| fileUrl | 是 | string | 应用安装文件链接（仅支持apk、xapk） | https://material.geelark.cn/user-upload/495622401615203830/apk/uCeztwvXpswsHPH5WFht.apk |
| desc | 否 | string | 备注 | 应用备注 |

## 请求示例

```json
{
    &quot;fileUrl&quot; : &quot;https://material.geelark.cn/user-upload/495622401615203830/apk/uCeztwvXpswsHPH5WFht.apk&quot;,
    &quot;desc&quot; : &quot;app description&quot;
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| taskId | string   | 任务id，用于查询上传状态  |

## 响应示例

```json
{
    &quot;traceId&quot;: &quot;B82825D58F85FB31B755BD6CA32A30A9&quot;,
    &quot;code&quot;: 0,
    &quot;msg&quot;: &quot;success&quot;,
    &quot;data&quot;: {
        &quot;taskId&quot;: &quot;1830906144634757120&quot;
    }
}
```

## 错误码

错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
