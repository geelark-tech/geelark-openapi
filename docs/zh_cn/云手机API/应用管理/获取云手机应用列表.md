## 接口说明

获取云手机已安装的应用列表

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/list`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| envId | 是 | string | 云手机id，对应获取云手机列表返回的id | 123456654321 |
| page        | 是     |    integer  | 页码，最小为1 | 1|
| pageSize    | 是     |    integer  | 每页多少条数据，最小为1，最大为100 | 10|

## 请求示例

```json
{
    "envId" : "1809135651036667904",
    "page" : 1,
    "pageSize" : 5
}
```

## 响应示例

```json
{
    "traceId": "123",
    "code": 0,
    "msg": "success",
    "data": {
        "items": [
            {
                "appIcon": "http://cmp1-prod.zxpcloud.com/apps/io.tm.k.drama/K-DRAMA_1716451323126.png",
                "appId": "1793552962123993090",
                "appName": "K-DRAMA",
                "appVersionId": "1793552962140770305",
                "installStatus": 1,
                "installTime": "2024-07-10 23:07:56",
                "packageName": "io.tm.k.drama",
                "versionCode": "21120300",
                "versionName": "1.0.1"
            }
        ],
        "total": 1,
        "page": 1,
        "pageSize": 5
    }

```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| total | integer   | 任务总数  |
| page | integer   | 页码  |
| pageSize | integer   | 分页大小  |
| items | array[AppInfo]   | 数据数组  |

### 应用信息 AppInfo

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| appIcon | string   | 应用图标 |
| appId | string    | 应用id |
| appName  | string    | 应用名 |
| appVersionId  | string    | 应用版本id |
| installStatus  | int    | 安装状态：0-正在安装，1-安装成功，2-安装失败，3-正在卸载，4-卸载成功，5-卸载失败，其他-未安装 |
| installTime  | string    | 安装时间 |
| packageName | string | 应用包名|
| versionCode | string | 应用版本码|
| versionName | string | 应用版本名|

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 42001 | 不存在对应的云手机 |
