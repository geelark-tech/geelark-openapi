## 接口说明

获取云手机可安装应用列表

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/installable/list`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| name | 否 | string | 搜索键值 | tiktok |
| envId | 是 | string | 云手机环境id | 123456654321 |
| getUploadApp | 否 | bool | 获取上传的应用 | true |
| page        | 是     |    integer  | 页码，最小为1 | 1|
| pageSize    | 是     |    integer  | 每页多少条数据，最小为1，最大为100 | 10|

## 请求示例

```json
{
    "name" : "tiktok",
    "envId" : "1809135651036667904",
	"getUploadApp" : false,
    "page" : 1,
    "pageSize" : 5
}
```


## 响应体数据说明

请求成功时，不返回数据。

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| total | integer   | 任务总数  |
| page | integer   | 页码  |
| pageSize | integer   | 分页大小  |
| items | array[AmpAppInfo]   | 数据数组  |

### 可安装应用信息 AmpAppInfo

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| appIcon | string   | 应用图标 |
| id | string    | 应用id |
| appName  | string    | 应用名 |
| packageName | string | 应用包名|
| appVersionInfoList | array[AmpAppVersionInfo] | 应用包名|

### 可安装应用版本信息 AmpAppVersionInfo
| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | string    | 应用版本id |
| installStatus  | int    | 安装状态：0-正在安装，1-安装成功，2-安装失败，3-正在卸载，4-卸载成功，5-卸载失败，其他-未安装 |
| versionCode | string | 应用版本码|
| versionName | string | 应用版本名|

## 错误码

| 错误码 | 说明 |
| --- | --- |
| 42001 | 不存在对应的云手机 |
