## 接口说明
查询云手机上传文件状态
上传文件任务请求后可在一小时内通过该接口主动获取结果，过期后将获取失败

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/uploadFile/result`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| taskId | 是 | string | 任务id | 参考请求示例 |


## 请求示例
```json
{
    &quot;taskId&quot;: &quot;528715748189668352&quot;
}
```



## 响应示例

```json
{
    &quot;traceId&quot;: &quot;A62BBBF3A294487F9B49B9FFA0F84CA8&quot;,
    &quot;code&quot;: 0,
    &quot;msg&quot;: &quot;success&quot;,
	&quot;data&quot;: {
        &quot;status&quot;: 1
    }
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| status | int | 0获取失败；1上传中；2上传成功；3上传失败 |


## 错误码

错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
