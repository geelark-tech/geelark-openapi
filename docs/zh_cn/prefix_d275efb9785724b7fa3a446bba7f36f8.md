## 接口说明

查询补丁安装状态，目前只支持Android 12系统、Android 13系统、Android 15系统

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/patch/install/status`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| type | 是 | int | 查询类型 | 1：根据patchId查询；2：根据taskId查询 |
| patchId | 否 | string | 补丁id，type为1必传 | 1830906144634757120 |
| id | 否 | string | 云手机id，type为1必传 | 1830906144634757120 |
| taskId | 否 | string | 任务id，type为2必传 | 1830906144634757120 |

## 请求示例

```json
{
 "type": 1,
 "patchId": "sdktest",
 "id": "532930887177274368"
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| status | int | 状态 0正在安装；1安装成功；2安装失败|

## 响应示例

```json
{
 "traceId": "B9C9A787A1B559A6B883A171A7EA129B",
 "code": 0,
 "msg": "success",
 "data": {
 "status": 1
 }
}
```

## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 42007 | 不存在对应的任务 |
