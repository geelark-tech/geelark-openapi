## 接口说明

请先调用自定义流程查询接口获取浏览器自定义任务流程，使用流程id创建任务

## 请求URL

- `http://localhost:40185/api/v1/browser/task/add`

## 请求方法

- POST

## 请求参数

| 参数名       | 必选   |     类型   |    说明    | 示例|
| ----------- | -------| -----------|----------- |----------- |
|eid|是|string|环境id|497652752864775437|
|name|否|string|任务名称，最多128字|myTask|
|remark|否|string|备注，最多200字|myRemark|
|scheduleAt|是|integer|计划时间，秒级时间戳|1741846843|
|flowId|是|string|任务流程id，任务流程查询接口返回的id字段|497652752864775437|
|paramMap|否|object|任务流程参数，文件类型要用数组|参考请求示例|


## 请求示例

```json
{
    "name":"test",
    "remark":"test remark",
    "scheduleAt": 1741846843,
    "eid":"557536075321468390",
    "flowId": "562316072435344885",
    "paramMap": {
        "Title": "video",
        "Desc": "this is video",
        "Video": ["https://material.geelark.cn/a.mp4"]
    }
}
```

## 响应示例

```json
{
    "traceId": "A4D8BCF69B878A71AC589F5CB1D80EAB",
    "code": 0,
    "msg": "success",
    "data": {
        "taskId": "558017255909123564"
    }
}
```
