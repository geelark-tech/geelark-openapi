## 接口说明

添加代理，重复的代理将不会添加

## 请求URL

- `https://openapi.geelark.cn/open/v1/proxy/add`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| list | 是 | array[ProxyAddItem] | 代理信息项列表，最多传100个 | 参考请求示例 |

### ProxyAddItem 代理信息项

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| scheme | 是 | string | 代理类型，socks5，http，https | socks5 |
| server | 是 | string | 代理地址 | 192.3.8.1 |
| port | 是 | integer | 代理端口 | 8000 |
| username | 否 | string | 代理用户名 | admin |
| password | 否 | string | 代理密码 | admin |

## 请求示例

```json
{
	"list": [
		{
			"scheme": "socks5",
			"server": "192.3.8.1",
			"port": 8000,
			"username": "admin",
			"password": "admin"
		}
	]
}
```

## 响应体数据说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| totalAmount | integer | 处理总数，和传入的代理信息项数量相同 |
| successAmount | integer | 添加成功数 |
| failAmount | integer | 添加失败数 |
| failDetails | array[FailDetail] | 失败代理信息 |
| successDetails | array[SuccessDetail] | 成功代理信息 |

### FailDetail 失败代理信息

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| index | integer | 传入的代理信息项索引 |
| code | integer | 错误码，参考失败细节错误 |
| msg| string | 错误信息，参考失败细节错误 |

### SuccessDetail 成功代理信息

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| index | integer | 传入的代理信息项索引 |
| id | string | 代理id，传入的代理信息项相同，则代理id相同 |

### 失败细节错误

| 错误码 | 错误信息 |
| - | - |
| 40000 | unknown error |
| 45003 | proxy not allow |
| 45004 | check proxy failed |
| 45007 | proxy already exists |
