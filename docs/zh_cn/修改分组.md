## 接口说明


- 修改某个分组信息包括 name, remark


## 请求 URL


- `https://openapi.geelark.cn/open/v1/group/update`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型         | 说明     | 示例         |
| ------ | ---- | ------------ | -------- | ------------ |
| list   | 是   | array[Group] | 分组数据 | 参考请求示例 |


### list 分组数据 <Group>


| 参数名 | 必选 | 类型   | 说明       | 示例         |
| ------ | ---- | ------ | ---------- | ------------ |
| id     | 是   | string | 分组 id    | 参考请求示例 |
| name   | 否   | string | 新分组名，最长50个字符   | 参考请求示例 |
| remark | 否   | string | 新分组备注，最长500个字符 | 参考请求示例 |


- 若携带 name，则 name 不能为空字符串


## 请求示例


```json
{
  "list": [
    {
      "id": "528995439832269824",
      "name": "newGroupRemark",
      "remark": "update remark"
    }
  ]
}
```


## 响应示例


```json
{
  "traceId": "B068F85E849B683291AE8ECCBB7B3DB9",
  "code": 0,
  "msg": "success",
  "data": {
    "totalAmount": 1,
    "successAmount": 1,
    "failAmount": 0
  }
}
```


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明             |
| ------ | ---------------- |
| 43030  | 分组名为空       |
| 43032  | 分组不存在       |
| 43035  | 不允许操作未分组 |
