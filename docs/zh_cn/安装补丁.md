## 接口说明

安装补丁，目前只支持Android 12系统、Android 13系统、Android 15系统

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/patch/install`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| apkURL | 是 | string | 应用安装文件链接（仅支持apk、xapk） | https://material.geelark.cn/user-upload/495622401615203830/apk/uCeztwvXpswsHPH5WFht.apk |
| id | 是 | string | 云手机id | 493188646250219877 |

## 请求示例

```json
{
 "apkURL" : "https://material.geelark.cn/user-upload/495622401615203830/apk/uCeztwvXpswsHPH5WFht.apk",
 "id" : "493188646250219877"
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| taskId | string   | 任务id，用于查询上传状态  |

## 响应示例

```json
{
 "traceId": "B82825D58F85FB31B755BD6CA32A30A9",
 "code": 0,
 "msg": "success",
 "data": {
 "taskId": "1830906144634757120"
 }
}
```

## 错误码

错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)
