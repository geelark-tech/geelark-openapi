## 接口说明

修改代理

## 请求URL

- `https://openapi.geelark.cn/open/v1/proxy/update`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| list | 是 | array[ProxyUpdateItem] | 代理信息项列表，最多传100个 | 参考请求示例 |

### ProxyUpdateItem 代理信息项

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 代理id | 493188072704313353 |
| scheme | 是 | string | 代理类型，socks5，http，https | socks5 |
| server | 是 | string | 代理地址 | 192.3.8.1 |
| port | 是 | integer | 代理端口 | 8000 |
| username | 否 | string | 代理用户名 | admin |
| password | 否 | string | 代理密码 | admin |

## 请求示例

```json
{
	"list": [
		{
			"id": "493188072704313353",
			"scheme": "socks5",
			"server": "192.3.8.1",
			"port": 8000,
			"username": "admin",
			"password": "admin"
		}
	]
}
```

## 响应体数据说明

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| totalAmount | integer | 处理总数，传入重复的id不计数 |
| successAmount | integer | 处理成功数 |
| failAmount | integer | 处理失败数 |
| failDetails | array[FailDetail] | 失败代理信息 |

### FailDetail 失败代理信息

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 代理id |
| code | integer | 错误码，参考失败细节错误 |
| msg| string | 错误信息，参考失败细节错误 |

### 失败细节错误

| 错误码 | 错误信息 |
| - | - |
| 40005 | proxy not found |
| 40000 | unknown error |
| 45003 | proxy not allow |
| 45004 | check proxy failed |
| 45007 | proxy already exists |
| 45008 | proxy type not allow |
