## 接口说明
添加素材


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/create`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| url | 是 | string | 请参考[上传临时文件到GeeLark](https://open.geelark.cn/api/upload-getUrl) | 参考请求示例|
| tagsId | 否 | array[string] | 标签id | 参考请求示例 |
| fileName | 否 | string | 文件名，最多200字符  | 参考请求示例 |


## 请求示例
```json
{
 "url" : "https://material.geelark.cn/client-img/banner0903_cn.gif",
 "tagsId" : ["569577509402731994","569577514586891738"],
 "fileName":"a.jpg"
}
```


## 响应示例

```json
{
 "traceId": "ADD9A7489BB2198DBC0FB37082684CB0",
 "code": 0,
 "msg": "success",
 "data": {
 "id": "570606523940605924",
 "failDetails": [
 {
 "id": "5695775094027319941",
 "code": 43022,
 "msg": "tag not found"
 }
 ]
 }
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- 
| id | string | 素材id |
| failDetails | failDetails | 失败信息 |

### failDetails 失败信息 <FailDetails>

| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 错误码       |
| id           | string     | 标签 id      |
| msg          | string     | 错误信息     |

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 60001 | 超出素材库最大容量 |
| 60003 | 非法的链接，需要使用上传到GeeLark的资源链接 |
| 60004 | 文件格式不支持 |
| 43022  | 标签不存在 |
| 40005  | 资源不存在，请检查url的资源是否可用 |
