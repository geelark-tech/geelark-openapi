## 接口说明
删除素材


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/del`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| ids | 是 | array[string] | 素材id | 参考请求示例 |


## 请求示例
```json
{
    "ids" : ["569569510948864567"]
}
```


## 响应示例

```json
{
    "traceId": "A1A84FBC88912B5A9F77B681B2A9A983",
    "code": 0,
    "msg": "success",
    "data": {
        "totalAmount": 1,
        "successAmount": 0,
        "failAmount": 1,
        "failDetails": [
            {
                "id": "5695695109488645671",
                "code": 60005,
                "msg": "material not found"
            }
        ]
    }
}
```

## 响应体数据说明


| 参数名        |     类型           |     说明     |
| ------------- | ------------------ | ------------ |
| totalAmount   | integer            | 请求删除总数 |
| successAmount | integer            | 成功总数     |
| failAmount    | integer            | 失败总数     |
| failDetails   | array[FailDetails] | 失败部分信息 |


### failDetails 失败信息 <FailDetails>


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 错误码       |
| id           | string     | 标签 id      |
| msg          | string     | 错误信息     |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明       |
| ------ | ---------- |
| 60005  | 素材不存在 |
