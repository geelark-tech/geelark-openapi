## 接口说明

- 隐藏云手机应用的辅助服务
- 目前支持Android 12、13、15机型
- 会覆盖旧的配置

## 请求URL

- `https://openapi.geelark.cn/open/v1/phone/hideAccessibility`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| ids | 是 | array[string] | 云手机id数组 | ["599257164413959866"] |
| pkgName | 是 | array[string] | app包名数组 |  ["com.zhiliaoapp.musically"] |

## 请求示例

```json
{
	"ids": ["599257164413959866"],
	"pkgName" : ["com.zhiliaoapp.musically"]
}
```

## 响应体数据说明 
### failDetails 
| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | integer   | 云手机id  |
| code | integer   | 错误码  |
| msg | string   | 错误信息  |

## 响应示例

```json
{
	"traceId": "A17A45A3B3A49AB5A1BDB654B7C82B81",
	"code": 0,
	"msg": "success",
	"data": {
		"failDetails": [
			{
				"id": "599257164413959866",
				"code": 42001,
				"msg": "env not found"
			}
		]
	}
}
```

## 错误码

以下为接口特定错误码，其他错误码请参考[接口调用说明](https://open.geelark.cn/api/request-instructions)

| 错误码 | 说明 |
| --- | --- |
| 42001 | 不存在对应的云手机 |
| 43037 | 暂不支持该机型 |
