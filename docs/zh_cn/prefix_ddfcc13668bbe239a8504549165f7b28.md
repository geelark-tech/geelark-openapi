## 接口说明
添加素材标签


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/tag/create`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| name | 是 | string | 标签名（最多30字符） | 参考请求示例 |
| color | 否 | int | 标签颜色 0白 1红 2蓝 3绿 4黄 5紫 | 参考请求示例 |


## 请求示例
```json
{
    "name" : "test",
    "color" : 1
}
```


## 响应示例

```json
{
    "traceId": "95D85F2085A6C847B2D68ABAAD591997",
    "code": 0,
    "msg": "success",
    "data": {
        "id": "570461738663674391"
    }
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- 
| id | string | 标签id |

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 60002 | 标签名重复 |
