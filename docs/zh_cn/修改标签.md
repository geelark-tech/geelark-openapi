## 接口说明


- 修改标签信息包括 name, color


**标签颜色列表参考创建标签接口**


## 请求 URL


- `https://openapi.geelark.cn/open/v1/tag/update`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型       | 说明         | 示例         |
| ------ | ---- | ---------- | ------------ | ------------ |
| list   | 是   | array[Tag] | 修改标签数组 | 参考请求示例 |


### list 修改标签数组 <Tag>


| 参数名 | 必选 | 类型   | 说明       | 示例         |
| ------ | ---- | ------ | ---------- | ------------ |
| id     | 是   | string | 标签 id    | 参考请求示例 |
| name   | 否   | string | 新标签名，最长30个字符   | 参考请求示例 |
| color  | 否   | string | 新标签颜色 | 参考请求示例 |


- 若标签信息中携带 name, 则 name 不可为空字符串


## 请求示例


```json
{
  "list": [
    {
      "id": "528989565894198272",
      "name": "tagEmptyUpdate",
      "color": "red"
    },
    {
      "id": "528994200482481152",
      "name": "tagUpdate",
      "color": "red"
    }
  ]
}
```


## 响应示例


```json
{
  "traceId": "B178151A9586E88195C4BF1493BBC98B",
  "code": 0,
  "msg": "success",
  "data": {
    "totalAmount": 2,
    "successAmount": 2,
    "failAmount": 0
  }
}
```


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明           |
| ------ | -------------- |
| 43020  | 标签名字为空   |
| 43022  | 标签不存在     |
| 43023  | 标签颜色不支持 |
