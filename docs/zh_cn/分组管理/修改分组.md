## 接口说明


- 修改某个分组信息包括 name, remark


## 请求 URL


- `https://openapi.geelark.cn/open/v1/group/update`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型         | 说明     | 示例         |
| ------ | ---- | ------------ | -------- | ------------ |
| list   | 是   | array[Group] | 分组数据 | 参考请求示例 |


### list 分组数据 &lt;Group&gt;


| 参数名 | 必选 | 类型   | 说明       | 示例         |
| ------ | ---- | ------ | ---------- | ------------ |
| id     | 是   | string | 分组 id    | 参考请求示例 |
| name   | 否   | string | 新分组名，最长50个字符   | 参考请求示例 |
| remark | 否   | string | 新分组备注，最长500个字符 | 参考请求示例 |


- 若携带 name，则 name 不能为空字符串


## 请求示例


```json
{
  &quot;list&quot;: [
    {
      &quot;id&quot;: &quot;528995439832269824&quot;,
      &quot;name&quot;: &quot;newGroupRemark&quot;,
      &quot;remark&quot;: &quot;update remark&quot;
    }
  ]
}
```


## 响应示例


```json
{
  &quot;traceId&quot;: &quot;B068F85E849B683291AE8ECCBB7B3DB9&quot;,
  &quot;code&quot;: 0,
  &quot;msg&quot;: &quot;success&quot;,
  &quot;data&quot;: {
    &quot;totalAmount&quot;: 1,
    &quot;successAmount&quot;: 1,

  }

```


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明             |
| ------ | ---------------- |
| 43030  | 分组名为空       |
| 43032  | 分组不存在       |
| 43035  | 不允许操作未分组 |
