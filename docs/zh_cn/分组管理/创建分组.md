## 接口说明


- 创建指定**名字，备注（可选）**的分组
- 支持批量


## 请求 URL


- `https://openapi.geelark.cn/open/v1/group/add`


## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型             | 说明         | 示例         |
| ------ | ---- | ---------------- | ------------ | ------------ |
| list   | 是   | array[GroupItem] | 分组数据列表 | 参考请求示例 |


### list 数据列表&lt;GroupItem&gt;


| 参数名 | 必选 | 类型   | 说明     | 示例   |
| ------ | ---- | ------ | -------- | ------ |
| name   | 是   | string | 分组名，最长50个字符   | tag    |
| remark | 否   | string | 分组备注，最长500个字符 | remark |


## 请求示例


```json
{
  "list": [
    {
      "name": "group"
    },
    {
      "name": "groupRemark",
      "remark": "remark"
    },
    {
      "name": "groupInvalid"
    }
  ]
}
```

## 响应体数据说明


| 参数名         |     类型              |     说明     |
| -------------- | --------------------- | ------------ |
| totalAmount    | integer               | 请求创建总数 |
| successAmount  | integer               | 成功总数     |
| failAmount     | integer               | 失败总数     |
| successDetails | array[SuccessDetails] | 成功部分信息 |
| failDetails    | array[FailDetails]    | 失败部分信息 |


### successDetails 成功信息 &lt;SuccessDetails&gt;


| 参数名       |     类型   |     说明       |
| ------------ | ---------- | -------------- |
| id           | string     | 分组 id        |
| name         | string     | 分组名         |
| remark       | string     | 无备注时不显示 |


### failDetails 失败信息 &lt;FailDetails&gt;


| 参数名       |     类型   |     说明     |
| ------------ | ---------- | ------------ |
| code         | integer    | 失败码       |
| name         | string     | 失败分组名   |
| msg          | string     | 失败信息     |


## 错误码


以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)


| 错误码 | 说明         |
| ------ | ------------ |
| 43030  | 分组名为空   |
| 43031  | 分组名已存在 |
