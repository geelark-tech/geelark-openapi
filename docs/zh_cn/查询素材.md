
## 接口说明
查询素材


## 请求URL

- `https://openapi.geelark.cn/open/v1/material/search`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| page | 否 | int | 页码 | 参考请求示例 |
| pageSize | 否 | int | 每页条数（最大200） | 参考请求示例 |
| fileName | 否 | string | 搜索文件名 | 参考请求示例 |
| tagsId | 否 | array[string] | 搜索标签id | 参考请求示例 |
| source | 否 | int | 来源 0-上传，1-AI剪辑 2百度云盘 3GhostCut 4GoogleDrive 5图生视频 | 参考请求示例 |
| fileType | 否 | array[int] | 文件类型 1-图片，2-视频 | 参考请求示例 |
| ids | 否 | array[string] | id数组，数组最大长度为100 | ["5213214343124321"] |


## 请求示例
```json
{
    "fileType" : [1],
    "fileName" : "demo",
    "tagIds" : ["569577514586891738"],
    "source" : 1,
    "page" : 1,
    "pageSize" : 50,
	"ids": ["608127302118696420"]
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- 
| total | int | 总条数 |
| page | int |当前页码 |
| pageSize | int | 当页条数 |
| list | list | 素材数据 |

## list 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- 
| id | int | 素材id |
| createdTime | int |创建时间，秒级时间戳 |
| fileName | string | 素材名 |
| fileSize | int | 素材大小 byte |
| fileUrl | string | 素材链接 |
| fileType | int | 素材类型  1-图片，2-视频 |
| width | int | 宽 |
| height | int | 高 |
| source | int | 来源 0-上传，1-AI剪辑 2百度云盘 3GhostCut 4GoogleDrive 5图生视频 |
| tags | tags | 标签数据 |
| userName | string | 上传用户名 |

## tags响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- 
| id | int | 标签id |
| name | int |标签名 |
| color | int | 标签颜色 0白 1红 2蓝 3绿 4黄 5紫 |
