## 接口说明

用于关闭对应的浏览器，需要指定环境ID。


## 请求URL

- `http://localhost:40185/api/v1/browser/stop`

## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 浏览器id | 539893235657500146 |


## 请求示例


```json
{
 "id": "539893235657500146"
}
```


## 响应示例


```json
{
    "traceId": "123456ABCDEF",
    "code": 0,
    "msg": "success"
}
```


## 错误码


以下为接口特定错误码，其他错误码请参考[浏览器错误码](https://open.geelark.cn/api/browser-error-codes)

| 错误码 | 说明 |
| --- | --- |
| -1 | 关闭失败 |
| 43028 | 子用户没有该环境组权限 |
