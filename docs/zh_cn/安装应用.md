## 接口说明

安装应用

## 请求URL

- `https://openapi.geelark.cn/open/v1/app/install`

## 请求方法

- POST

## 请求参数

| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| envId | 是 | string | 云手机环境id | 1809135651036667904 |
| appVersionId | 是 | string | 应用版本id | 1793552962140770305 |

## 请求示例

```json
{
    "envId" : "1809135651036667904",
    "appVersionId" : "1793552962140770305"
}
```


## 响应示例

```json
{
    "traceId": "123",
    "code": 0,
    "msg": "success"
}
```

## 错误码

以下为接口特定错误码，其他错误码请参考[云手机错误码](https://open.geelark.cn/api/cloud-phone-error-codes)

| 错误码 | 说明 |
| --- | --- |
| 42001 | 不存在对应的云手机 |
| 42002 | 云手机不处于运行状态 |
| 42003 | 正在安装对应的app |
| 42004 | 已安装高版本应用，不允许安装低版本 |
| 42006 | 不存在对应的app |
