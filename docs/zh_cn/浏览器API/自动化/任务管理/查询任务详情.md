## 接口说明

获取浏览器任务详情

## 请求URL

- `http://localhost:40185/api/v1/browser/task/detail`

## 请求方法

- POST

## 请求参数

| 参数名       | 必选   |     类型   |    说明    | 示例|
| ----------- | -------| -----------|----------- |----------- |
| taskId | 是 | string| 任务id|497652752864775437 |


## 请求示例

```json
{
    "taskId": "497652752864775437"
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
|id|string|任务id|
|eid|string|环境id|
|name|string|任务名|
|remark|string|任务备注|
|serialName|string|环境名|
|status|integer|任务状态，1-等待执行 2-执行中 3-任务完成 4-任务失败 7-任务取消|
|startAt|integer|开始时间，秒级时间戳|
|finishAt|integer|结束时间，秒级时间戳|
|cost|integer|耗时秒数|
|resultCode|integer|任务结果码，参考任务失败代码及原因|
|resultDesc|string|任务结果描述，参考任务失败代码及原因|
|scheduleAt|integer|任务计划执行时间，秒级时间戳|
|logs|array[string]|任务日志|

### 任务失败代码及原因

| 失败代码 | 失败原因 |
| - | - |
|20001|任务的开始时间早于当前时间|
|20002|浏览器启动失败|
|29999|任务被中断|
