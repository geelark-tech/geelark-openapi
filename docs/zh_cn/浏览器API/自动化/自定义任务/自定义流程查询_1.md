## 接口说明

获取浏览器自定义任务流程列表

## 请求URL

- `http://localhost:40185/api/v1/browser/task/flow`

## 请求方法

- POST

## 请求参数

| 参数名       | 必选   |     类型   |    说明    | 示例|
| ----------- | -------| -----------|----------- |----------- |
| page | 否 | integer| 页码，不填默认为1|1 |
| pageSize |否 | integer| 页大小，最大为100，不填默认为10| 10|


## 请求示例

```json
{
    &quot;page&quot;: 1,
	&quot;pageSize&quot;: 10
}
```

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
|total|integer|数据总数|
|page|integer|页码|
|pageSize|integer|页大小|
|list|array[Flow]|数据列表|

### 任务流程 Flow

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
|id|string|任务流程id|
|title|string|任务流程标题|
|desc|string|任务流程备注|
|params|array[string]|任务流程参数字段名|
