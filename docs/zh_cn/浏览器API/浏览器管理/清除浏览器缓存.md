## 接口说明

清除指定浏览器环境的本地缓存。为了数据安全，请确保在该浏览器环境未打开时使用该接口。若磁盘空间不足，可使用该接口清除缓存释放空间。


## 请求URL

- `http://localhost:40185/api/v1/browser/deleteCache`

## 请求方法


- POST






## 请求参数


| 参数名 | 必选 | 类型 |  示例 | 说明
| --- | --- | --- | --- | --- |
| ids | 是 | array[string] | ["123456789","987654321"] |浏览器id|
| type | 是 | array[string] | ["local_storage","indexeddb"] | 清除缓存的类型。支持：local_storage,indexeddb,extension_cache,cookie,history,image_file。 |


## 请求示例


```json
{
    "ids": [
        "123456789",
        "987654321"
    ],
    "type": [
        "local_storage",
        "indexeddb",
        "extension_cache",
        "cookie",
        "history",
        "image_file"
    ]
}
```


## 响应示例


```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "success_count": 2,
        "error_count": 0,
        "error_info": []
    }
}
```
