## 接口说明

用于检查指定浏览器的启动状态，需指定环境ID


## 请求URL

- `http://localhost:40185/api/v1/browser/status`

## 请求方法


- POST






## 请求参数


| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 浏览器id | 539893235657500146 |


## 请求示例


```json
{
 "id": "539893235657500146"
}
```


## 响应示例


```json
{
  "code": 0,
  "msg": "success",
  "data":{
    "status":"open",	//浏览器已打开运行中 "open" ，未打开则是 "close"
	"debugPort": 11000
  }
}
```
