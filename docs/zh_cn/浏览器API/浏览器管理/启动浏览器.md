## 接口说明

用于启动浏览器，需要指定环境ID。

## 请求URL

- `http://localhost:40185/api/v1/browser/start`

## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| id | 是 | string | 浏览器id | 539893235657500146 |


## 请求示例


```json
{
 "id": "539893235657500146"
}
```


## 响应示例


```json
{
  "traceId": "123456ABCDEF",
  "code": 0,
  "msg": "success",
  "data":{
  	"debugPort": 11000
  }
}
```





## 错误码


以下为接口特定错误码，其他错误码请参考[浏览器错误码](https://open.geelark.cn/api/browser-error-codes)


| 错误码 | 说明 |
| --- | --- |
| -1 | 启动失败 |
| 43007 | 环境有其他人在使用 |
| 43008 | 打开环境数达到上限 |
| 46003 | 该环境不在套餐内 |
| 43028 | 子用户没有该环境组权限 |
| 90002 | 该环境不存在 |
| 90003 | 磁盘空间不足 |
