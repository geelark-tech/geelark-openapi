## 接口说明

删除不需要的环境，支持批量删除，一次删除的数量不能超过100个。


## 请求URL

- `http://localhost:40185/api/v1/browser/delete`

## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| envIds | 是 | array[string] | 浏览器id数组，最多100个 | [“539893235657500146”] |


## 请求示例


```json
{
  "envIds": ["539893235657500146"]
}
```


## 响应示例


```json
{
	"traceId":"123456ABCDEF",
	"code":0,
	"msg":"success",
	"data":{
		"busyIds":["539893235657500146"],
		"serverErrIds":["539893235657500147"],
		"successIds":["539893235657500148"]
	}
}
```


## 响应体数据说明


| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| successIds | array[string] | 删除成功的环境id |
| busyIds | array[string] | 正在使用中的环境id |
| serverErrIds | array[string] | 处理异常的环境id |


## 错误码

| 错误码 | 说明 |
| --- | --- |
| 43028 | 子用户没有该环境组权限 |
