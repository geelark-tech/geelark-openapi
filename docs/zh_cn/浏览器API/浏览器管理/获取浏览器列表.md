## 接口说明

查询已创建的环境信息，包含代理信息、代理ID等。


## 请求URL

- `http://localhost:40185/api/v1/browser/list`

## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型 | 说明 | 示例 |
| --- | --- | --- | --- | --- |
| page | 否 | int | 页码，最小为1，不填默认为1 | 1 |
| pageSize | 否 | int | 每页多少条数据，最小为1，最大为100，不填默认为10 | 10 |
| ids | 否 | array[string] | 环境id，最大为100个，为空才使用分页参数 | [“5213214343124321”] |
| serialName | 否 | string | 环境名称 | myEnv |
| remark | 否 | string | 备注 | myRemark |
| groupName | 否 | string | 分组名 | myGroup |
| tags | 否 | array[string] | 标签名称数组 | [&quot;myTag&quot;] |


## 请求示例


```json
{
 &quot;page&quot;: 1,
 &quot;pageSize&quot;: 10
}
```

## 响应体数据说明


| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| total | int | 数据总数 |
| page | int | 当前页码 |
| pageSize | int | 每页多少条数据 |
| items | array[BrowserApiSearchSimpleItem] | 数据列表 |

### 浏览器数据 BrowserApiSearchSimpleItem

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 环境id |
| serialName | string | 环境名称 |
| serialNo | string | 环境编号 |
| group | EnvGroup | 环境分组 |
| remark | string | 备注 |
| tags | array[EnvTag] | 环境标签 |
| proxy | EnvPhoneListProxy | 代理信息 |
| accountInfo | BrowserApiSearchSimpleAccount | 账号信息 |
| simulateInfo | BrowserApiSearchSimpleSimulate | 仿真信息 |

### 环境分组 EnvGroup

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| id | string | 环境分组id |
| name | string | 环境分组名 |
| remark | string | 环境分组备注 |

### 环境标签 EnvTag

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| name | string | 标签名 |

### 代理信息 EnvPhoneListProxy

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| type | string | 代理类型 |
| server | string | 代理主机 |
| port | int | 代理端口 |
| username | string | 代理用户名 |
| password | string | 代理密码 |

### 账号信息 BrowserApiSearchSimpleAccount

| 参数名 | 类型 | 说明 |
| --- | --- | --- |
| url | string | 平台地址 |
| userName | string | 平台账号 |
| passWord | string | 平台密码 |
| afterStartup | int | 启动后打开页面，1-恢复上次访问，2-打开指定url，3-打开指定url同时打开平台页面，4-恢复上次访问同时打开平台页面 |
| autoOpenUrls | array[string] | 指定url |
