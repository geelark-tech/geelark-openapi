## 接口说明

用于将环境重新分组，通过ID将环境分到对应的组。


## 请求URL

- `http://localhost:40185/api/v1/browser/moveGroup`

## 请求方法


- POST


## 请求参数


| 参数名 | 必选 | 类型 |  示例 | 说明
| --- | --- | --- | --- | --- |
| envIds | 是 | array[string] | ["123456789","987654321"] |浏览器id，最多100个|
| groupId | 是 | string | "123456789" |分组id，传”0”则移动至未分组|


## 请求示例


```json
{
    "envIds":["123456789","987654321"],
    "groupId":"0"
}
```


## 响应示例


```json
{
    "traceId": "123456ABCDEF",
    "code": 0,
    "msg": "success",
}
```
