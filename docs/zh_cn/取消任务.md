## 接口说明

取消浏览器任务，等待执行或执行中可以取消

## 请求URL

- `http://localhost:40185/api/v1/browser/task/cancel`

## 请求方法

- POST

## 请求参数

| 参数名       | 必选   |     类型   |    说明    | 示例|
| ----------- | -------| -----------|----------- |----------- |
| taskId | 是 | string| 任务id|497652752864775437 |


## 请求示例

```json
{
    "taskId": "497652752864775437"
}
```

## 响应示例

```json
{
	"traceId":"2XsBK1HDR",
	"code":0,
	"msg":"success"
}
```

## 错误码

| 错误码 | 说明 |
| --- | --- |
|48001|任务状态不允许操作|
|48005|只允许创建者操作|
