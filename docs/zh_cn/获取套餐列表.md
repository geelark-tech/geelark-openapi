## 接口说明

获取套餐列表

## 请求URL

- `https://openapi.geelark.cn/open/v1/pay/profiles/list`

## 请求方法

- POST

## 响应体数据说明

| 参数名       |     类型   |    说明    |
| ----------- | -----------|----------- |
| id | string   | 套餐id |
| price  |  float   | 套餐一个月价格 |
| level | int   |套餐等级 0Base 1Pro |
| envNum | int   |套餐环境数 |
| freeTime | int   |套餐免费分钟数 |
| openEnvNumOneDay | int   | 每日打开数 |
| createEnvNumOneDay | int   | 每日创建数 |
